# ESport Tournament API

# Requirements/Technologies
-	NodeJS 10+
-   Javascriptt
-   Typescript
-   Jest
-   Express

# Installation
-  Clone `git@github.com:devpilot-official/tribally-api.git`
-  Run `yarn install`
-  Add environment variables
-  Run `yarn build`

# Run
-   Development `yarn start:dev`
-   Production `yarn start:prod`
